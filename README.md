# Fantasy Akadémia (kickoff)

Ez a repository most a **1. fázis indító struktúrát** tartalmazza a Firebase-alapú matematika gyakorló alkalmazáshoz.

## Mappák
- `app/`: React + Tailwind frontend
- `functions/`: Firebase Cloud Functions (TypeScript)
- `docs/phase1-kickoff.md`: döntések, terv, séma, route-ok, function lista

## Következő lépés
- Blokkoló auth döntések véglegesítése után implementálható a teljes PIN login flow és tanulóimport.
